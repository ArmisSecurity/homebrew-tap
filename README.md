# Armis Homebrew Tap

Official Homebrew tap for Armis tools.

## Installation

### armis-cli

Enterprise-grade CLI tool for static application security scanning with Armis Cloud.

```bash
brew tap armissecurity/tap
brew install armis-cli
```

Or install directly:

```bash
brew install armissecurity/tap/armis-cli
```

## Available Formulas

- **armis-cli** - Static application security scanning CLI

## Documentation

For more information about armis-cli, visit the [official repository](https://github.com/ArmisSecurity/armis-cli).
